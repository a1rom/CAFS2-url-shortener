# URL Shortner

## General Info

The app creates short links

## Launch
To run this project, insall it locally using npm:
```
$ npm i
```
create css and font files via laravel-mix:
```
$ npm run prod
```

Make sure the server points to /public folder

## Technologies:

Project is created with:
* PHP
* Bootstrap/Sass
* Laravel Mix
